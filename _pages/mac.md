## 符號代表的特定按鍵

| 符號 | 按鍵             |
| ---- | ---------------- |
| ⌘    | Command          |
| ⇧    | Shift            |
| ⌥    | Option           |
| ⇪    | Caps Lock        |
| ⌃    | Control          |
| ↩    | Return           |
| ⌫    | Backspace/Delete |
| ⌦    | Forward Delete   |
| ↑    | 向上鍵           |
| ↓    | 向下鍵           |
| ←    | 向左鍵           |
| →    | 向右鍵           |
| ⇞    | Page Up          |
| ⇟    | Page Down        |
| ↖    | Top (Home)       |
| ↘    | End              |
| ⎋    | Escape (Esc)     |
| ␣    | Space            |
| ⏏    | Eject            |

[Mac 上選單中的符號代表什麼？](https://support.apple.com/zh-tw/guide/mac-help/cpmh0011/mac)

## 鍵盤佈局與輸入

在 Mac 上，按一下選單列上的「輸入法」選單，然後選擇「顯示模擬鍵盤」。

若該指令沒有顯示，請選擇「蘋果」選單 >「系統偏好設定 ⋯」，按一下「鍵盤」，按一下「輸入方式」，然後選取「在選單列中顯示輸入法選單」。
